### Архивация и восстановление данных
- [Borg Backup](https://borgbackup.readthedocs.io/en/stable/) - Удобный инструмент для резервного копирования данных. [![Open-Source Software][OSS Icon]](https://borgbackup.readthedocs.io/en/stable/development.html) ![Freeware][Freeware Icon]
- [Deja Dup](https://www.linux.com/learn/total-system-backup-and-recall-deja-dup) - Простой инструмент резервного копирования со встроенным шифрованием. [![Open-Source Software][OSS Icon]](https://launchpad.net/deja-dup) ![Freeware][Freeware Icon]
- [Duplicity](http://duplicity.nongnu.org/) - Создаёт зашифрованные tar-архивы и отправляет их на удалённый или локальный сервер. [![Open-Source Software][OSS Icon]](https://launchpad.net/duplicity) ![Freeware][Freeware Icon]
- [Photorec](http://www.cgsecurity.org/wiki/PhotoRec) - Программа для восстановления потерянных файлов (видео, документов, архивов, фотографий) с жёстких дисков, CD-ROM и карт памяти. ![Freeware][Freeware Icon]
- [Qt4-fsarchiver](https://sourceforge.net/projects/qt4-fsarchiver/) - Графическая оболочка для fsarchiver (сохранение/восстановление разделов, папок, MBR/GPT) для Debian, Suse и Fedora. [![Open-Source Software][OSS Icon]](https://borgbackup.readthedocs.io/en/stable/) ![Freeware][Freeware Icon]
- [rclone](http://rclone.org/) - Командная утилита для синхронизации файлов и каталогов с различными облачными хранилищами, поддерживает зашифрованные бэкапы. [![Open-Source Software][OSS Icon]](https://github.com/ncw/rclone) ![Freeware][Freeware Icon]
- [rsnapshot](http://rsnapshot.org/) - Утилита на основе rsync для создания периодических снимков локальных/удалённых машин, использует жёсткие ссылки для экономии дискового пространства. [![Open-Source Software][OSS Icon]](https://github.com/rsnapshot/rsnapshot.git) ![Freeware][Freeware Icon]
- [System Rescue CD](http://www.system-rescue-cd.org/SystemRescueCd_Homepage) - Загрузочный диск или USB-флешка для администрирования и восстановления системы после сбоя. ![Freeware][Freeware Icon]
- [Test Disk](http://www.cgsecurity.org/wiki/TestDisk) - Мощное бесплатное ПО для восстановления потерянных разделов и восстановления загрузки дисков при повреждении ПО. ![Freeware][Freeware Icon]
- [Timeshift](https://launchpad.net/timeshift) - Утилита для восстановления системы, создаёт инкрементальные снимки с помощью rsync и жёстких ссылок, которые можно восстановить позже. ![Freeware][Freeware Icon]

## Как помочь
> Просто укажите **название** приложения в списке.
> Дайте ссылку на его **домашнюю страницу** или руководство по установке.
> Напишите **краткое описание** и добавьте **иконку**.
> Убедитесь, что оно помещено в **соответствующий раздел**.
> Если приложение не подходит ни под один существующий раздел, создайте **новый**.
> Соблюдайте **алфавитный порядок**.

*Пункты, отмеченные ![Open-Source Software][OSS Icon], имеют открытый исходный код и ссылку на код. Пункты с ![Freeware][Freeware Icon] бесплатны (как бесплатное пиво).*

Если у вас есть проблемы с приложениями, пожалуйста: -> обратитесь на страницу разработчика (если есть) -> создайте issue там -> создайте issue здесь, чтобы я мог рассмотреть, стоит ли удалить приложение из списка.
